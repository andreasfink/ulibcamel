3.	Content
--------------

This SDL specificaiton forms the normative behaviour of the
INAP CS-2. An introduction to the modelling approach can be
found in Annex A, section 1 of Q.1228.

Two alternative versions of the SDL diagrams are provided:

   Diskette 1: inap.zip
               (contains SDL files in zipped SDT format)

               This SDL specification in SDT format is best
               viewed with the font settings Arial 8 for
               Headings and Arial 6 for symbol texts.

               Annex A.8 SDLs (CS2-SCF) are only available
               in PDF format.

Diskettes 2 and 3 contain the same SDL files in PDF format:

   Diskette 2: cs1_inap.pdf  (Annex A.3)
               cs2_inap.pdf  (Annex A.4)
   Diskette 3: cs2_srf.pdf   (Annex A.5)
               assisting.pdf (Annex A.6)
               cs2_cusf.pdf  (Annex A.7)
               cs2_scf.pdf   (Annex A.8)
